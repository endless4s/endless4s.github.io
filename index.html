    endless is a Scala library to describe event sourced entities using tagless-final algebras, running with built-in implementations for Akka.

    /ˈɛndləs/: having or seeming to have no end or limit

    endless refers to the ever flowing stream of events capturing state evolution with event sourcing, and is a nod to tag-less.
