final class RepositoryT[F[_], S, E, ID, Alg[_[_]]] extends Repository[F, ID, Alg]

RepositoryT[F, S, E, ID, Alg] is a data type implementing the Repository[F, ID, Alg] entity access ability.

It assembles the client command protocol together with the command router natural transformation on OutgoingCommand[*] to deliver an instance of Alg[F] allowing to interact with the specific entity in the cluster.

For the server-side it provides capability of running an incoming command via interpretation of the algebra with EntityT.
